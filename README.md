# Aplikasi CRUD Mahasiswa
Aplikasi data mahasiswa menggunakan Java, Hibernate, dan JasperReport

## Fitur
- Create, Read, Update, Delete (CRUD) data mahasiswa
- Koneksi database menggunakan Hibernate ORM
- Cetak Laporan dengan JasperReports
- Export Laporan ke PDF

## Teknologi
- Java Swing
- Hibernate ORM
- MySQL
- JasperReports

## Database
Menggunakan database dengan nama db_mahasiswa yang memiliki tabel dengan nama mahasiswa dimana struktur tabelnya terdapat id, nama, npm, dan jurusan
## Setup
1. Clone / download project Java
2. Import library berikut:
- Hibernate Core
- MySQL Connector/J
- JasperReports
- Jaspersoft / iReport
3. Konfigurasi Hibernate di file hibernate.cfg.xml
4. Sesuaikan username & password MySQL
5. Jalankan aplikasi melalui NetBeans / IDE lainnya
